# YouTube Revenue Intelligence: Predicting Channel Growth with Machine Learning

## 📖 Overview
This project analyzes real-world YouTube channel performance data to identify the factors influencing revenue, audience engagement, and subscriber growth.

Using Data Analytics and Machine Learning techniques, the project predicts estimated revenue and extracts actionable insights for content creators, marketers, and analysts.

---

## 🚀 Key Features

✔ Cleaned and transformed 70+ performance metrics  
✔ Performed Exploratory Data Analysis (EDA)  
✔ Visualized revenue, engagement & growth trends  
✔ Built Random Forest Regression Model  
✔ Predicted Estimated Revenue (USD)  
✔ Generated feature importance insights  
✔ Business recommendations for creators

---

## 📊 Dataset Includes

- Views & Watch Time  
- Likes, Shares, Comments  
- Subscribers & Returning Viewers  
- Ad Impressions  
- CPM / Revenue Metrics  
- Thumbnail CTR  
- Audience Retention Metrics

---

## 🛠 Tech Stack

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-Learn  
- Jupyter Notebook

---

## 🤖 Machine Learning Model

Model Used:

- Random Forest Regressor

Evaluation Metrics:

- RMSE
- R² Score

---

## 📈 Key Insights

- Higher engagement strongly impacts revenue  
- Views + CTR directly influence monetization  
- Returning viewers improve channel growth  
- Upload timing affects performance
